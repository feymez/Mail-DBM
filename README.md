Aby wszystko działało musisz pierw utworzyć kilka ról. Nazwy daj jakie chcesz ważne abyś je rozpoznawał. Będziesz potrzebować:
  - Mail NO
  - Mail OFF
  - Mail
  - Zalogowany
  - Wylogowany
