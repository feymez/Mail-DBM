# Mail-DBM

## System mailowy Discord Bot Maker ~ feymez <br>

Aby wszystko działało musisz pierw utworzyć 5 ról. Nazwy daj jakie chcesz ważne abyś je rozpoznawał. Będziesz potrzebować: <br>

  - Mail OFF
  - Mail
  - Zalogowany
  - Wylogowany
